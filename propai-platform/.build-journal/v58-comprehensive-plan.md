## 2. 관련 국제 표준 및 프레임워크 분석

### 2.1 EVM (Earned Value Management) — PMBOK 7th Edition

**국제 표준**: PMI PMBOK 7th (2021), ANSI/EIA-748 (EVMS), ISO 21508:2018
**핵심 수학식**:
- EV (Earned Value) = BAC × 실제공정률(%)
- SV (Schedule Variance) = EV - PV → 양수=앞서감, 음수=지연
- CV (Cost Variance) = EV - AC → 양수=절감, 음수=초과
- CPI (Cost Performance Index) = EV / AC
- SPI (Schedule Performance Index) = EV / PV
- EAC (Estimate at Completion) = BAC / CPI

**핵심 논문**:
- Wauters & Vanhoucke (2024) — ML 기반 EVM 예측: LSTM/GBM이 기존 EAC 대비 MAPE 8.2% 달성 (기존 15.4%)
- Narbaev & De Marco (2023) — Earned Schedule 기반 회귀 모델: EAC 정확도 15-23% 향상
**한국 법규**: 건설기술진흥법 시행령 제59조, 조달청 대형공사(500억+) EVMS 의무
**v58 구현**: `supervision_service.py` — Phase 22
**구현 고려사항**: 현장 사진 AI 분석(OpenCV)으로 공정률 자동 산출 → EVM 자동화

### 2.2 ISO 31000:2018 리스크 관리

**국제 표준**: ISO 31000:2018 (원칙/가이드라인), ISO 31010:2019 (31개 리스크 평가 기법)
**핵심 수학식**: Risk = P(가능성) × I(영향도) × E(노출도)
**등급화**: critical(>0.6) / high(>0.4) / medium(>0.2) / low(≤0.2)
**핵심 논문**:
- Kim & Lee (2023, KSCE) — BERT 기반 리스크 자동 등급화: 한국 건설 420건 데이터, 정확도 91.3%
- Chen & Hsu (2024) — Monte Carlo 10,000회 NPV VaR 95% 신뢰구간 투자 의사결정
**한국 법규**: 금융위원회 PF 리스크 관리 가이드라인(2024), 건설산업기본법 시행규칙 제30조
**v58 구현**: `risk_service.py` — Phase 29
**10대 리스크 매트릭스**: 허가지연, 공사비초과, 공정지연, 시장침체, 금리상승, 자재수급, 안전사고, 법규변경, 민원, 자연재해

### 2.3 LCA ISO 14040:2006 — 전과정 평가

**국제 표준**: ISO 14040:2006/Amd1:2020, ISO 14044:2006/Amd2:2020, EN 15978:2011, EN 15804:2012+A2:2019
**핵심 수학식**: GWP_total = Σ(m_i × EF_i × CF_i)
**4단계**: A1-A3(자재생산) → B1-B7(사용) → C1-C4(폐기) → D(재활용)
**IPCC AR6 2021 GWP-100**: CO2=1, CH4=27.9, N2O=273 | 콘크리트C25=0.159, 철근=1.460, 구조강재=1.770
**핵심 논문**:
- Hoxha et al. (2023) — NLP 기반 BIM 재료↔EPD 자동 매칭: 정확도 94.7%, 저탄소 대체재 GWP 평균 23% 감소
- Huang & Gao (2024) — AI 저탄소 자재 추천: Pareto Front 다목적 최적화(GWP+비용+구조성능)
**한국 법규**: G-SEED 2023 LCA 배점 상향, 탄소중립기본법, 환경부 탄소성적표지
**v58 구현**: `lca_service.py` — Phase 13

### 2.4 LCC ISO 15686-5:2017 — 생애주기비용

**국제 표준**: ISO 15686-5:2017, ISO 15686-1:2011, EN 15459-1:2017, ASTM E917
**핵심 수학식**: LCC = Σ(C_t / (1+d)^t), d = 실질할인율
**실질할인율**: d_real = (1+d_nominal)/(1+inflation) - 1
**교체비용 주기**: 15년/25년/35년/45년 (건설비의 5%)
**핵심 논문**:
- Marzouk & Azab (2023) — BIM-LCC 통합 자동화: 에너지 비용이 전체 LCC의 35-42%
- Lee et al. (2024) — LSTM 유지보수비 예측: 한국 아파트 150개소 10년 이력 RMSE 3.2M원/동
**한국 법규**: 조달청 500억+ 공공건축물 LCC 분석 의무, 국토부 LCC 가이드(할인율 4.5%, 40년)
**v58 구현**: `lcc_service.py` — Phase 15

### 2.5 EPD ISO 21930:2017 — 환경제품선언

**핵심 수학식**: CF_material = Σ(m_i × EPD_i)
**EPD Korea 데이터베이스**: 14종 자재 탄소계수 내장
**특이사항**: CLT 구조목 = -0.690 kgCO2e/kg (탄소 저장 효과 → 음수값)
**v58 구현**: `epd_carbon_service.py` — Phase 20f

### 2.6 ZEB — 녹색건축물 조성 지원법 제17조

**핵심 수학식**: EAR(에너지자립률) = E_renewable / E_total × 100 (%)
**등급**: ZEB 1등급(100%↑) ~ ZEB 5등급(20%↑)
**시뮬레이션**: EnergyPlus IDF (eppy 라이브러리)

### 2.7 EOQ — 경제적 주문량 모델

**핵심 수학식**: EOQ = √(2DS/H)
**D=연간수요량, S=1회주문비용, H=연간재고유지비용**
**한국은행 PPI 연동**: 8개 자재별 생산자물가지수 기반 최적 발주시기 예측
**v58 구현**: `procurement_optimizer.py` — Phase 38p

### 2.8 자연재해 리스크 — 자연재해대책법

**핵심 수학식**: Risk_score = Σ(w_i × H_i × E_i × V_i)
**H=재해빈도(행정안전부 재해연보), E=노출도(강까지 거리), V=취약도(층수)**
**가중치**: 홍수(0.4), 산사태(0.3), 지진(0.3)
**v58 구현**: `disaster_risk_service.py` — Phase 38o

## 3. 스토리라인 수립

### 3.1 플랫폼 전체 내러티브

PropAI는 **부동산 개발사업의 전주기**(기획→설계→인허가→시공→분양→입주→운영→자산관리)를
**AI가 자동화하는 통합 플랫폼**이다.

사용자(시행사/건설사/디벨로퍼)가 **부지 PNU 코드**를 입력하면:

```
[1단계] 부지 분석
  PNU → VWORLD API → GIS Union 경계 통합 → 용도지역 자동 조회
  → 인접 시세 AVM(XGBoost R²=0.94) → 입지 적합성 점수(스마트시티 6지표)

[2단계] 법규 검토
  40개 법령 AI 자동 검증(ALRIS RAG + FAISS)
  → 건폐율/용적률/높이제한/이격거리/일조권 자동 산출
  → 법규 위반 시 자동 보정

[3단계] 설계 생성
  CNN 참조이미지(ResNet-50) 기반 설계 자동 생성
  → 배치도(SVG) + 평면도(DXF) + 3D 조감도(Three.js)
  → 법규 자동 보정 CAD 파라메트릭 편집(ezdxf)

[4단계] 사업성 분석
  Monte Carlo 10,000회 시뮬레이션
  → NPV/IRR/투자수익률 확률 분포
  → 민감도 분석(변수별 영향도)
  → 리스크 등급화(ISO 31000)

[5단계] ESG 산출
  LCA(ISO 14040) 탄소 자동 계산 (IPCC AR6)
  + LCC(ISO 15686-5) 생애주기비용 최적화
  + ZEB 에너지자립률 시뮬레이션(EnergyPlus)
  + RE100/K-ETS 탄소배출권 산출
  + EPD(ISO 21930) 자재별 탄소발자국
  + G-SEED/EU Taxonomy 인증 적합성

[6단계] 인허가
  세움터 API 자동 연동 → 건축 인허가 신청
  + 종상향/지구단위계획 자동 분석
  + 분양가상한제 자동 검증
  + 법규 변경 실시간 감지(법제처 API)

[7단계] 시공 관리
  착공 체크리스트 자동 생성(건축법 제21조)
  + 안전관리계획서 자동 작성(건설기술진흥법 제62조)
  + EVM 공정/원가 관리(PMBOK)
  + AI 사진 분석 공정률 자동 산출
  + BIM/IFC 물량 산출 + DXF 도면 자동 생성
  + 자재 조달 최적화(EOQ) + PPI 연동

[8단계] 운영 관리
  디지털 트윈 실시간 운영(IFC 4.3)
  + IoT 센서 에너지 AI 최적화(HVAC)
  + 생애주기 부품 교체 최적화(ISO 15686-1)
  + 포트폴리오 통합 KPI 관리
  + 자연재해 리스크 GIS 분석
```

### 3.2 사용자 여정 (User Journey)

```
[회원가입/로그인] → [대시보드] → [새 프로젝트 생성]
     ↓
[부지 선택 (지도)] → [PNU 입력 / 지도 클릭]

     ↓

     ↓
```
